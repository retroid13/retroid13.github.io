Game Boy Advance

In Japan, on March 21, 2001, Nintendo released a significant upgrade to the Game Boy line. The Game Boy Advance (also referred to as GBA) featured a 32 bit 16.8 MHz ARM.

Key features:

  • 2.9 inch TFT color LCD
  • Stereo headphone jack
  • No backlight
  • Powered by 2 AA batteries

Acessories

There is a plethora of accessories for GBA, both official and unofficial and this is very common among any console made by Nintendo.

Hardware Modifications

Backlight

One of the most common and wanted modifications is the backlit mod using the screen from an AGS-101 (the Gameboy Advance SP).
